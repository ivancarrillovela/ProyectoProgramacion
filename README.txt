# Aplicación Voluntariado Cuatrovientos

Antes de ejecutar la aplicación es necesario ejecutar abrir y ejecutar el archivo 'BBDD_Proyecto' en SQL Server
ya que es la query que creara nuestra BBDD.

Una vez hecho esto es necesario que vayamos al archivo 'conexión.txt' situado en \AppProgramacion\Aplicacion Voluntariado\Aplicacion Voluntariado\bin\Debug\ y escribamos de forma única el nombre de nuestro servidor SQL Server.

Una vez ya realizados estos pasos podemos ejecutar la aplicación sin problema.
